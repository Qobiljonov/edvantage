# Adtage — Milliy AI Ta'lim Platformasi

O'zbekiston uchun premium AI ta'lim ekotizimi.

## Ishga tushirish

```bash
npm install
npm run dev
```

## Dizayn tizimi

- **Tipografiya:** Inter, qattiq tracking (-0.03em display)
- **Ranglar:** Ink (#0B1220), Brand blue, Gold (milliy prestij), Teal
- **Sirtlar:** `surface`, `surface-elevated`, `surface-inset`
- **Animatsiya:** Apple easing `cubic-bezier(0.16, 1, 0.3, 1)`
